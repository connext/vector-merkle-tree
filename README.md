# connext/vector-merkle-tree

A merkle tree implementation tailored for Connext Vector transfers. Written
in Rust and compiled to WebAssembly for use in Vector server and browser
nodes.

## Building

```sh
npm run prepare
```

## Publishing

```sh
npm version ...
npm publish
```
